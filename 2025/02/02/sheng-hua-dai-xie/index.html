生物化学代谢

2025-02-02 生物 711 字 · 约 2 分钟

生物化学课程代谢反应的特征总结，从产物、过程等特征来总结相关反应式。

  • 生成PPi的反应：

    • 生成UDP-G（活性葡萄糖，UTP脱PPi）
    • 脂酸 + CoA + ATP -> 脂酰~CoA + AMP + PPi
    • 氨基酸 + ATP -> 氨基酰~AMP + PPi（氨基酸的活化）
    • 脂酰CoA的形成（脂肪酸的活化）
    • CDP-含氮碱的合成（CTP脱PPi）
    • 瓜氨酸+天冬氨酸 -> 精氨酸代琥珀酸
    • 嘌呤碱合成第一步
    • 嘧啶碱合成第五步
    • DNA复制5′至3′的聚合
    • 两段DNA之间的连接（DNA复制）
    • 氨基酰-tRNA的合成（翻译）
    • 尿素合成总方程式

  • 底物水平磷酸化：

    • 1,3-二磷酸甘油醛 -> 3-磷酸甘油醛（磷酸甘油酸激酶，糖酵解）
    • 磷酸烯醇式丙酮酸 -> 丙酮酸（丙酮酸激酶，糖酵解）
    • 琥珀酰CoA -> 琥珀酸（琥珀酰CoA合成酶，TCA）

  • 重要反应循环：

    • 糖酵解
    • TCA
    • 糖异生
    • 脂肪酸代谢
    • 氨基酸代谢
    • 尿素循环

  • 不可逆反应：

    • G -> G-6-P（糖酵解1）
    • F-6-P的磷酸化成F-1,6-P（糖酵解3）
    • PEP生成丙酮酸（糖酵解10）
    • 丙酮酸生成乙酰CoA（有氧部分第一步）
    • 乙酰CoA与草酰乙酸生成柠檬酸（TCA1）
    • α-酮戊二酸生成琥珀酰CoA（TCA4）
    • 延胡索酸生成苹果酸（TCA7）

  • 氧化脱羧：

    • 丙酮酸  → 乙酰CoA
    • 异柠檬酸生成α-酮戊二酸（TCA3）
    • α-酮戊二酸生成琥珀酰CoA（TCA4）

  • FAD参与的反应：

    • 琥珀酸生成延胡索酸（TCA6）
    • 脂酰CoA脱氢生成烯脂酰CoA（β氧化1）
